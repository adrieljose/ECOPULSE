# EcoPulse - Air Quality Monitoring System

EcoPulse is a real-time air quality monitoring dashboard designed to provide high-density, actionable environmental data.

## Features

- **Real-time AQI Tracking**: Monitor air quality indices in real-time.
- **Unified Dashboard**: A high-density "App Bar" for quick access to device and time filters.
- **Dark Mode**: Eye-friendly interface for low-light environments.
- **Responsive Design**: Works across desktop and mobile devices.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5.
- **Backend**: PHP.
- **Data Source**: Python script for serial-to-API communication.

## Setup

1. Clone the repository.
2. Host it on a local PHP server (e.g., Laragon).
3. Connect your monitoring device via serial.
4. Run the `serial_to_api.py` script.
